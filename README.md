ВАСАП, это мои 2 pipeline для 2 соревнований на kaggle   
Основной pipeline лежит в папке > 'default pipeline' он копируется в новые проекты
## **ЗАПУСК**  
перейдите в папку соревнования, далее в config.py. **Сделать настройку**:
 - selectedmodels - отвечает за выбранные модели в обучении,
 - selectedensemble - отвечает за выбранный/выбранные ансамбли | mlp не работает в ансамбле,
 - train - включает обучение модели,
 - use_submit - делать ли submit,
 - FE - Feature Engineering,
 - savemodel - сохранение модели, в dl только веса,
 - param_on - включает загрузку параметров,
 - optuna.on - включение поиска параметров через optuna,
 - ensemble.on - включает ансамбли,
 - NN.on - включает поддержку mlp.
 - !!! Если у вас не nvidia установите параметр device = None в NN <br>
